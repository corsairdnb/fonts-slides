Советы на все времена

  • Используйте только нужные unicode-глифы.
  • Используйте правильные форматы шрифтов каждого браузера.
  • Ставьте http-заголовки для кеширования.
  • Избегайте конвертирования.
  • Используйте GZIP.

Font Face Observer

  • Требуется подключить скрипт ~ 1.5 kb (gzip)
  • Шрифты подключаются любым способом

Font Face Observer

			var font = new FontFaceObserver('My Family');
			 
			font.load().then(function () {
			  console.log('Font is available');
			}, function () {
			  console.log('Font is not available'>);
			});

Font Face Observer

  • асинхронность
  • браузерный кеш
  • подключение любым способом
  • промисы и таймауты
  • отсутствие FOUT / FOIT
