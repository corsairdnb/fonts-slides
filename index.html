Что делает браузер:

  1. запрашивает HTML-документ
  2. анализирует HTML и создает модель DOM
  3. находит ссылки на CSS и отправляет запросы
  4. создает модель CSSOM, совмещая ее с DOM и получает модель отрисовки
  5. определяет необходимые варианты шрифтов, отправляет запросы
  6. отображает страницу
  7. если шрифт ещё не доступен, браузер может не показывать текст
  8. как только шрифт доступен, браузер показывает текст

Советы на все времена

  • Используйте только нужные unicode-глифы.
  • Используйте правильные форматы шрифтов каждого браузера.
  • Ставьте http-заголовки для кеширования.
  • Избегайте конвертирования.
  • Используйте GZIP.
